# Visual Studio Code Extensions

Here is my extension list, also a scale of usage (1-5) meaning how often I use it.

## Editor

These extensions is generally for text editing that increase editing workflow, and not language specific.

| Extensions                                                                                                     | Scale of usage (1-5) | Description                                                                                                                                      |
| -------------------------------------------------------------------------------------------------------------- | -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| [Better Comments](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)                  | ⭐⭐⭐⭐             | Gives different color highlight for different comment type                                                                                       |
| [Advanced New File](https://marketplace.visualstudio.com/items?itemName=patbenatar.advanced-new-file)          | ⭐⭐⭐⭐⭐           | Create new file on a specific folder just by keystroke, I mapped `⌘ + N` to this extension                                                       |
| [Bracketeer](https://marketplace.visualstudio.com/items?itemName=pustelto.bracketeer)                          | ⭐⭐⭐⭐             | Useful to change bracket, or remove bracket, select text inside a quote, change quote (", ', \`)                                                 |
| [Dracula Theme](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula)               | Default Theme        | Best dark theme for me. I even use it on my website                                                                                              |
| [File Utils](https://marketplace.visualstudio.com/items?itemName=sleistner.vscode-fileutils)                   | ⭐⭐⭐⭐⭐           | A convenient way of creating, duplicating, moving, renaming and deleting files and directories                                                   |
| [GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)                                 | ⭐⭐                 | A git extension that show git status on the line                                                                                                 |
| [Indent Rainbow](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow)                   | ⭐⭐⭐⭐             | Makes indentation easier to read. You can only use bracket color, but I prefer this one                                                          |
| [Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)           | Default Icons        | Great icons, it has a lot of icons for folders too                                                                                               |
| [Case Preserve](https://marketplace.visualstudio.com/items?itemName=Cardinal90.multi-cursor-case-preserve)     | ⭐⭐⭐               | This one is super useful, when you select occurences (`⌘ + D`) and they have different cases, it will preserve the case when you change the text |
| [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)                         | ⭐⭐⭐⭐⭐           | This extension is for formatting code, it will format the code when you save it                                                                  |
| [Tab Out](https://marketplace.visualstudio.com/items?itemName=albert.TabOut)                                   | ⭐⭐⭐               | Escape bracket using tabs, similar to eclipse editor                                                                                             |
| [VS Sequential Number](https://marketplace.visualstudio.com/items?itemName=neptunedesign.vs-sequential-number) | ⭐⭐⭐               | Input sequential number in the across multiple cursors. This is unexpectedly useful                                                              |
